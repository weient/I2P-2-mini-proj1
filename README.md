## 程式設計二：mini project1
### building a small computer
#### project概要：模擬compiler將程式語言翻譯成組合語言(assembly code)的過程
* input: a list of expressions, consisting of:
  * intergers
  * operators
  * three initial variables x, y, z
  * some new variables
  
* ouput: a list of assembly code
